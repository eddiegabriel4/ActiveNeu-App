# ActiveNeu
ActiveNeu is an iOS app designed for iPhone that lets you analyze various activity metrics, as well as motivate you to be more active by adding two social features. That allow you to create and join custom group leaderboards of these metrics, as well as how you stack up against everyone using the app.
